HƯỚNG DẪN SỬ DỤNG

1. Mở file index.html để xem website.
2. Danh sách đề nằm ngay ở mục: Chọn đề để xem trực tiếp.
3. Nhấn vào tên đề để xem PDF trong khung bên dưới.
4. Có thể dùng ô tìm kiếm hoặc lọc theo tỉnh/thành.
5. Thư mục de-thi chứa toàn bộ đề PDF.

Lưu ý: Khi đưa lên Google Sites, chỉ cần nhúng trang index.html sau khi lưu trữ website ở nơi có thể truy cập công khai.
